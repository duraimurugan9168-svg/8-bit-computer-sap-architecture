# 8-bit-computer-sap-architecture

## Introduction

SAP-8 is an FPGA-based 8-bit computer architecture inspired by the Simple-As-Possible (SAP) computer model. The project is implemented entirely in Verilog HDL and is designed to demonstrate the fundamental concepts of computer organization, including instruction fetching, decoding, execution, and data processing.

The architecture consists of modular components such as the Program Counter (PC), Memory Address Register (MAR), Instruction Register (IR), Accumulator Register, B Register, Arithmetic Logic Unit (ALU), Control Unit, RAM, and Output Register. These modules work together through a common bus system to execute instructions stored in memory.

The primary objective of this project is to provide a simple yet extensible platform for learning digital design, computer architecture, and FPGA implementation. The design supports arithmetic and logical operations and can be further enhanced with advanced features such as branching instructions, stack operations, interrupts, and peripheral interfaces.

This project serves as an educational tool for students and enthusiasts interested in processor design and hardware description languages.
